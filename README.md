# Text-Message

A simple android SMS app.

**App**

![APP GIF](APP_GIF.gif)

---

## Features

- Used Kotlin to code out the app.
- Uses Androids send, recieve and write functionality. 
- Allows users to view messages.
- Has setting for auto-reply function.
